# shop-web

Frontend application for the `React Demo Shop` project.

## Run

```bash
npm install
npm run dev
```

Dev server: `http://localhost:5173`

## Build

```bash
npm run build
npm run preview
```

## Stack

- React + TypeScript + Vite
- React Router
- Redux Toolkit
- TanStack Query
- React Hook Form + Zod
- Tailwind CSS

See root documentation for full project setup: [`../README.md`](../README.md)
